### Skills to Know About
- `/statusline` — Configures the status bar shown at the bottom of Claude Code. Shows model, context usage, git worktree, and more. Run it once when you first set up Claude Code.
- `/agents` — Opens the agents dialog to browse and manage available subagents.
- `/review` — Kicks off a code review of staged or recent changes. Use it before opening a PR.
- `/effort` — Adjusts how much time/tokens Claude spends on a response. Useful when you want a quick answer vs. a thorough one.
- `/init` — Bootstraps a `CLAUDE.md` for a new repo. Run this when onboarding a codebase that doesn't have one yet.
- `/model` — Switches the active model mid-session.
